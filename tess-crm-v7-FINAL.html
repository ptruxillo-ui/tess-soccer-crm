⚠️ Quick Setup Required (One Time)
Because this CRM is a local HTML file, you need to register a free Microsoft app to enable the connection. This takes about 15 minutes and is 100% free forever. Full step-by-step guide below.

📋 Setup Steps

  1. Go to Azure Portal → App Registrations
  2. Click "New registration"
  3. Name: Tess Soccer CRM
  4. Supported account types: "Personal Microsoft accounts only"
  5. Redirect URI: Select "Single-page application (SPA)" → Enter: http://localhost
  6. Click "Register"
  7. Copy the Application (client) ID from the overview page
  8. Go to "API permissions" → Add permission → Microsoft Graph → Delegated → Add: Mail.Read, Mail.Send
  9. Click "Grant admin consent"
  10. Paste your Client ID below and click Connect
💡 Your Client ID is stored locally in this browser only. Microsoft uses it to securely connect your Outlook account.
